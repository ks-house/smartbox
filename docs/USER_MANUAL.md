# ☀️ 태양광 스마트 자동 수거함 통합 사용자 매뉴얼

본 매뉴얼은 태양광 충전 시스템과 초음파/BLE 센서를 결합한 스마트 수거함의 안전한 조립, 운영 및 유지보수를 위한 가이드입니다.

## 1. 시스템 구성 부품

* **메인 제어부:** ESP32-C6 (초음파 제어, BLE 무선 통신, Wi-Fi 스마트홈 연동)
* **센서부:** 방수 초음파 센서 (AJ-SR04T)
* **구동부:** 12V 선형 액추에이터 (Linear Actuator), 1채널 릴레이(메인 전원 제어), 2채널 릴레이(모터 방향 제어)
* **전원부:** 태양광 패널, 태양광 충전 컨트롤러, 12V 납축전지

## 2. 단계별 조립 순서

1. **전원부 결선:** 태양광 패널을 충전 컨트롤러에 연결한 뒤, 컨트롤러를 12V 납축전지와 연결합니다.
2. **구동부 설치:** 수거함 뚜껑 경첩과 본체 사이에 12V 선형 액추에이터를 단단히 고정합니다.
3. **제어부 배선:** ESP32-C6 보드에 1채널 및 2채널 릴레이를 연결합니다. *(배선 중에는 메인 전원 스위치를 반드시 OFF로 둡니다.)*
4. **센서 부착:** 수거함 전면 중앙에 방수 초음파 센서를 장착하고 제어부로 선을 빼내어 연결합니다.

## 3. 🚨 핵심 경고 사항 (안전 수칙)

* **하드웨어 인터락:** 모터의 정회전/역회전(뚜껑 개폐)을 전환하기 전, 반드시 1채널 릴레이를 통해 12V 메인 전원을 먼저 차단해야 합니다. 이를 어길 시 배터리 단락(쇼트) 및 화재의 위험이 있습니다.
* **작업 시 전원 차단:** 내부 배선을 점검하거나 수거함을 분해/조립할 때는 물리적인 메인 전원 스위치를 완전히 끄십시오.

## 4. 운영 시나리오

### 🚶‍♂️ 일상 출입 (초음파 및 BLE)

* **초음파 감지 모드:** 수거함 정면 50cm 이내로 접근하면 자동으로 뚜껑이 개방됩니다. 뚜껑이 열린 후 5초간 대기하며, 사용자가 멀어지면 자동으로 닫힙니다.
* **BLE 연동 모드:** 등록된 스마트폰이나 BLE 태그를 소지한 사용자가 접근하면, 별도의 제스처 없이도 시스템이 이를 인식하여 뚜껑을 부드럽게 엽니다.

### 🛠️ 관리자 모드

* 쓰레기통을 비우거나 내부를 청소할 때, 시스템을 '관리자 모드'로 전환하면 뚜껑이 닫히지 않고 열린 상태로 고정됩니다.

### 🌐 네트워크 및 스마트홈 연동

* 내부 Wi-Fi를 통해 가정 내 스마트홈 플랫폼(SmartThings, Home Assistant 등)과 연동할 수 있습니다.
* 앱을 통해 수거함의 뚜껑 상태를 원격으로 제어하거나, 태양광 배터리 잔량 및 충전 상태를 모니터링할 수 있습니다.

## 5. 실시간 모니터링 및 유지보수

* **실시간 센서 수치 확인:** 유지보수 시 콘솔이나 웹 인터페이스를 통해 초음파 센서의 거리 측정값(cm)을 실시간으로 확인하여 센서의 상태를 점검할 수 있습니다.
* **메인 전원 스위치 운영:** 장기간 수거함을 사용하지 않거나(예: 장기 외출) 태양광 충전이 불가능한 악천후가 지속될 경우, 메인 전원 스위치를 내려 배터리 방전을 예방합니다.

## 6. 비상조치 및 문제 해결 가이드

* **뚜껑이 열리지 않거나 반응이 없을 때:**
1. 메인 전원 스위치가 `ON` 위치에 있는지 확인합니다.
2. 스마트홈 앱을 통해 12V 납축전지의 전압이 너무 낮지 않은지(방전 여부) 확인합니다.


* **사람이 없는데 뚜껑이 혼자 움직일 때:**
초음파 센서 표면에 이물질이나 빗물이 맺혀 있는지 확인하고 마른 수건으로 닦아줍니다.
* **시스템 먹통 (프리징) 발생 시:**
물리적 메인 전원 스위치를 `OFF`로 내린 뒤, 약 10초 후 다시 `ON`으로 켜서 시스템을 강제 재부팅(하드 리셋)합니다.
