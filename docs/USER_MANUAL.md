# ☀️ 태양광 스마트 자동 수거함 통합 사용자 매뉴얼

본 매뉴얼은 태양광 충전 시스템과 초음파 센서를 결합한 스마트 수거함의 안전한 조립, 운영, 웹 어드민 사용 및 비상 대응을 위한 가이드입니다.

---

## 1. 시스템 구성 부품

* **메인 제어부:** ESP32-C6 (초음파 제어, 비동기 웹 어드민 서버, Wi-Fi 및 SmartThings 연동)
* **센서부:** 방수 초음파 센서 (AJ-SR04T), INA219 전압/전류 모니터링 모듈
* **구동부:** 12V 선형 액추에이터 (Linear Actuator), 1채널 릴레이 (메인 12V 전원 차단용), 2채널 릴레이 (모터 정/역회전 방향 제어용)
* **전원부:** 태양광 패널, Solar Charge Controller, 12V 납축전지

---

## 2. 단계별 조립 및 배선 요약

1. **전원부 결선:** 태양광 패널을 충전 컨트롤러에 연결한 후, 12V 납축전지를 컨트롤러 배터리 단자에 연결합니다.
2. **구동부 설치:** 수거함 뚜껑 경첩과 본체 사이에 12V 선형 액추에이터를 유격 없이 단단히 고정합니다.
3. **제어부 배선:** 상세 핀 맵에 따라 릴레이와 초음파 센서, INA219 모듈을 ESP32-C6에 결선합니다.
   * *자세한 회로도 및 결선 정보는 [배선 가이드](WIRING_GUIDE.md)를 참조하십시오.*
4. **글리치 차단 점검:** 최초 기동 시 모터가 오동작하여 튀는 것을 방지하기 위해 릴레이의 기본 제어핀 상태는 고임피던스(`INPUT`) 상태로 유지됩니다.

---

## 3. 🚨 핵심 하드웨어 안전 수칙

이 시스템은 고전류 모터와 기계식 릴레이를 제어하므로 아래 수칙을 반드시 엄수해야 합니다.

1. **릴레이 인터락 (Relay Interlock):** 정회전 릴레이(`GPIO 7`)와 역회전 릴레이(`GPIO 8`)를 동시에 `LOW`(ON)로 출력해서는 안 됩니다. 이는 12V 배터리 단락(쇼트)과 화재를 유발할 수 있습니다.
2. **전원 선-차단 후-방향 전환:** 모터 구동 방향을 전환할 때, 무조건 1채널 메인 릴레이(`GPIO 6`)를 먼저 차단(`INPUT` 고임피던스 상태)하고, 최소 100ms 가드 타임을 보낸 후 방향 릴레이를 스위칭해야 합니다. 방향 접점이 완전히 결합된 뒤 다시 100ms 대기를 거쳐 메인 릴레이를 통전합니다.
3. **고임피던스 대기 모드:** 뚜껑이 열려 대기하거나 닫힌 standby 상태에서는 모든 릴레이 제어 핀을 `INPUT`(고임피던스) 상태로 전환하여 누설 전류를 완전히 리셋하고 대기 전력을 0mA로 격리시킵니다.

## 4. 실시간 웹 대시보드 (Web Control Panel)

본 시스템은 모바일 및 PC에서 직접 접속하여 상태를 모니터링하고 설정을 변경할 수 있는 **비동기 웹 대시보드**를 지원합니다.

### 접속 방법
1. 모바일이나 PC의 Wi-Fi 목록에서 **`SmartBox-WiFi`** AP를 선택하여 연결합니다. (비밀번호 없음)
2. 브라우저를 열고 주소창에 **`http://192.168.4.1/`**을 입력하여 접속합니다.

### 주요 기능 및 UI 화면
* **실시간 텔레메트리 모니터링:** 
  * 🔋 **배터리 전압 (Battery Voltage):** 소수점 둘째 자리까지의 전압 수치를 표시합니다.
  * ⚡ **액추에이터 전류 (Actuator Current):** 모터가 소비하는 전류(mA)를 표시합니다.
  * 📏 **감지 거리 (Human Distance):** 5샘플 중간값 필터링(Median Filter)을 거친 노이즈가 제거된 감지 거리를 표시합니다.
  * ⏱️ **동작 경과 시간 (Active Time):** 현재 상태 전이 후 경과 시간을 실시간 초 단위로 나타냅니다.
* **원격 수동 제어:**
  * **원격 강제 열기 (Remote Force Open):** 물리적인 센서 감지 없이 웹 화면에서 즉시 뚜껑 개방 시퀀스를 가동합니다.
  * **비상 정지 해제 (Release Emergency Lock):** 과전류로 인해 락인된 비상 정지 상태를 Standby 상태로 수동 리셋합니다.
* **실시간 감도 및 안전 캘리브레이션:**
  * **감지 거리 임계값 (Detection Distance):** 초음파 센서 감지 반경을 cm 단위로 직접 변경합니다.
  * **뚜껑 열림 유지 시간 (Lid Hold Time):** 사람이 감지된 후 개방 상태를 유지하는 시간(ms)을 입력합니다.
  * **모터 과전류 임계값 (Stall Current):** 물체 협착이나 모터 부하 상승 시 차단할 전류 임계값(mA)을 입력합니다.
  * *모든 설정값은 저장 버튼 클릭 시 즉시 컨트롤러에 반영되고, ESP32의 비휘발성 플래시 영역(Preferences)에 영구 저장되어 재부팅 후에도 유지됩니다.*
* **동적 Wi-Fi 프로비저닝 (Network Configuration):**
  * 대시보드 내의 **"Network Configuration"** 카드에서 **[Scan Networks]**를 누르면 주변의 Wi-Fi 신호를 비동기 스캔합니다.
  * 검색된 Wi-Fi 목록 중 연결할 SSID를 선택하고 비밀번호를 입력한 뒤 **[Connect]**를 누르면 기기가 외부 인터넷(STA 모드)에 연결됩니다.
  * 입력된 Wi-Fi 정보는 ESP32 NVS(Preferences)에 안전하게 저장되어 재부팅 시에도 자동으로 Wi-Fi 망 접속을 재시도합니다.
  * 기기가 외부 인터넷에 성공적으로 접속하면 헤더의 Wi-Fi 상태가 `Online`으로 활성화되며, NAS Pull OTA를 직접 수행할 수 있습니다.
* **무선 펌웨어 업데이트 (HTTPS NAS Pull):**
  * 대시보드 하단에 **"Firmware Update (from NAS)"** 관리 기능이 내장되어 있습니다.
  * **NAS Pull 방식**: 기기에 고정된 Synology HTTPS NAS 경로(`[YOUR_NAS_URL]`)로부터 최신 펌웨어를 장치가 직접 받아와 설치하는 **Fetch & Update** 방식입니다.
  * **안전 인터락 작동**: 펌웨어 쓰기가 개시되는 즉시, 리니어 액추에이터의 오작동 및 쇼트 화재 방지를 위해 **모든 릴레이 전원이 차단**되며 센서 측정 및 뚜껑 개폐가 일시적으로 완전 정지(FSM 동결)됩니다.

## 5. 자동 안전 보호 시스템 (FSM States)

시스템은 상태 머신(FSM)을 통해 다음의 안전 가드를 자율적으로 제어합니다.

* **뚜껑 열림 대기 및 자동 닫힘 (`STATE_HOLD`):**
  * 뚜껑이 완전히 열린 후, 사용자가 센서 감지 범위를 벗어나더라도 바로 닫히지 않고 설정된 시간(기본값 10초, 10000ms) 동안 열린 상태를 유지합니다.
  * 10초의 대기 시간이 경과하는 중 사용자가 다시 감지 범위 내로 접근하면, 대기 타이머가 자동으로 초기화되어 사용자가 물러날 때까지 개방 상태를 계속 연장하여 유지합니다.
* **협착 방지 비상 정지 (`STATE_EMERGENCY_STOP`):**
  * 뚜껑이 열리거나 닫히는 도중(초기 300ms 기동 전류 구간 제외) 모터 전류가 설정된 과전류 임계값(기본 800mA)을 초과하면, 즉시 모터를 비상 정지하고 모든 릴레이를 `INPUT`(고임피던스) 상태로 고립시킵니다.
  * 비상 정지 상태가 활성화되면 웹 대시보드의 경고 배지가 켜지며, 대시보드의 "비상 정지 해제" 버튼을 누르거나 장치를 재부팅할 때까지 안전 모드가 유지됩니다.
* **저전압 배터리 보호 (`STATE_BATTERY_LOW_SHUTDOWN`):**
  * 배터리 전압이 11.3V 이하로 3초 연속 유지될 경우, 최후의 잔여 전력을 공급하여 뚜껑을 강제로 완전 개방(`OPEN_START` 루틴 가동)합니다. 이는 관리자가 배터리 충전 및 점검을 위해 수거함 내부로 접근하기 용이하도록 하기 위함입니다.
  * 완전 개방 후, 배터리의 추가 방전을 막기 위해 모든 릴레이 보드의 핀을 고임피던스 상태로 분리하고 보드를 절전 대기 상태(동결)로 진입시킵니다.
* **무선 펌웨어 업데이트 격리 보호 (`STATE_OTA_UPDATING`):**
  * NAS로부터 펌웨어 업데이트 요청을 감지하면 즉시 이 모드로 전환됩니다.
  * 이 과정에서 `GPIO 6, 7, 8`의 모든 출력 접점이 물리적인 고임피던스(`INPUT`) 모드로 바뀌어 액추에이터 전원(12V)이 완벽하게 차단되며, 펌웨어가 100% 안전하게 다운로드되어 플래시에 써질 때까지 어떤 센서 신호에도 반응하지 않고 대기 전력을 0mA로 리셋하여 쇼트 위험을 원천 격리합니다.

## 6. 비동기 스마트홈 연동 (SmartThings)

* 수거함의 상태가 전환될 때마다(예: 대기 -> 열림 -> 닫힘 -> 비상 정지) 상태 정보가 비동기 HTTP Webhook을 통해 지정된 외부 SmartThings REST API 주소로 자동 전송됩니다.
* 백그라운드 FreeRTOS 큐(Queue) 및 최하위 우선순위 태스크 구조로 설계되어 있어, Wi-Fi 네트워크 지연이나 유실이 발생하더라도 뚜껑 감지 및 모터 비상 차단과 같은 안전 제어 루프를 절대로 블로킹하지 않습니다.

## 7. 문제 해결 가이드 (Troubleshooting)

| 현상 | 원인 분석 및 해결 조치 |
| :--- | :--- |
| **뚜껑이 열리지 않고 웹 화면에 반응이 없음** | 1. 12V 전원 스위치와 메인 배터리 잔량을 테스터기로 점검하십시오.<br>2. 웹 대시보드 상태창에서 배터리 전압이 11.3V 이하로 떨어져 저전압 셧다운 상태(`Low Battery`)로 고정되어 있는지 확인하십시오. |
| **주변에 사람이 없는데 뚜껑이 혼자 계속 열림** | 1. 초음파 센서 렌즈 표면에 물기나 이물질이 묻어 있는지 닦아주십시오.<br>2. 웹 대시보드의 감도 설정 탭에서 '감지 거리 임계값'을 다소 낮게(예: 50cm -> 40cm) 조정하십시오. |
| **뚜껑이 조금 움직이다가 멈추고 웹 대시보드에 'Emergency' 경고가 표시됨** | 1. 수거함 뚜껑 경첩 부위에 이물질이 끼어 모터가 과부하 상태(Stall)가 되었는지 점검하십시오.<br>2. 대시보드의 설정 탭에서 '모터 과전류 임계값'을 현재 힌지 가중치에 맞춰 적정하게 상향 조정하고 "비상 정지 해제" 버튼을 누르십시오. |
| **웹 어드민 대시보드 페이지 로딩이 실패함** | 스마트폰의 Wi-Fi 설정에서 **`SmartBox-WiFi`**에 성공적으로 연결되었는지 확인하고, 게이트웨이 IP 주소(`192.168.4.1`)를 브라우저 주소창에 정확히 입력했는지 점검하십시오. |
